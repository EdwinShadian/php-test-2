## Тестовое задание для компании "Фьюче"
### Описание

REST API на языке PHP с фреймворком Laravel. Данные запросов и ответов принимает в формате JSON.

**Эндпоинты:**
* /api/v1/notebook
  * GET: получить список всех записей. Если добавить query page={номер страницы}, то записи можно получать по 10 штук.
  * POST: добавить запись. Требуемые аргументы: name, phone_number, email. Необязательные: company, birthdate, photo.
* /api/v1/notebook/{id}
  * GET: получить одну запись по ее id.
  * PATCH: изменить существующую запись. Аргументы: см. выше.
  * DELETE: удалить запись (реализовано "мягкое" удаление через deleted_at).

Тестирование производилось с помощью unit-тестов и вручную через Postman.

Описания методов можно получить по эндпоинту /api/documentation.

Swagger-документация сгенерирована с использованием пакета [darkaonline\l5-swagger](https://github.com/DarkaOnLine/L5-Swagger).
